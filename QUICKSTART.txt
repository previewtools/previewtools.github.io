TROUBLESHOOTING:
=================

Problem: Page shows plain HTML without styles
Solution: Check that css/ folder exists with both CSS files

Problem: Slider not working
Solution: Ensure js/slider.js is present in correct path

Problem: Components not loading (blank header/footer)
Solution: Check that components/ folder exists with all 4 HTML files

Problem: Images not showing
Solution: Ensure images/ folder has logo.jpg and favicon.jpg

Problem: Page still loading after 5 minutes
Solution: Check GitHub Pages settings (should be on main/gh-pages branch)

Problem: Links give 404 errors
Solution: Ensure 404.html exists in root directory

Problem: SEO issues
Solution: Check og:url and og:image tags have correct domain
